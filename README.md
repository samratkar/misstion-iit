# IIT JEE Preparation: Mission IIT

Welcome to the Mission IIT preparation repository. This project is designed to help students master Physics, Chemistry, and Mathematics through **Question-Based Pedagogy (QBP)**.

## Core Philosophy
Instead of passive reading, students engage with carefully sequenced questions that:
1.  **Elicit Curiosity:** Start with a problem that challenges existing intuition.
2.  **Build Concepts:** Guide the student through the derivation and application of core principles.
3.  **Ensure Mastery:** Provide varying levels of difficulty to solidify understanding.

## Structure
- `/syllabus`: Detailed breakdown of JEE Main & Advanced topics.
- `/subjects`: The main question bank organized by subject and chapter.
- `/references`: Recommended books and resources.
- `/tests`: Sample mock papers and concept-specific quizzes.

## How to use this repository
1.  Pick a topic from the `syllabus`.
2.  Navigate to the corresponding directory in `subjects/`.
3.  Work through the `concepts-through-questions.md` files sequentially.
