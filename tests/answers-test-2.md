# Answer Key & Explanations: Sample Test 2

## Physics
1. **Answer: B ($u \cos \theta$)**
   - At the highest point, vertical velocity $v_y = 0$.
   - Horizontal velocity remains constant $v_x = u \cos \theta$.

2. **Answer: A (12 min)**
   - Shortest time $t = d/v_{br}$.
   - $t = 1 \text{ km} / 5 \text{ km/h} = 0.2 \text{ hours} = 12 \text{ minutes}$.

## Chemistry
3. **Answer: A ($8:27$)**
   - $r \propto n^2/Z$.
   - $r_1 (Li^{2+}, n=2) = 2^2/3 = 4/3$.
   - $r_2 (He^{+}, n=3) = 3^2/2 = 9/2$.
   - Ratio $= (4/3) / (9/2) = 8/27$.

4. **Answer: A ($4/3R$)** *Correction: Rydberg formula is $1/\lambda = R(1/n_1^2 - 1/n_2^2)$.*
   - For Lyman $1^{st}$ line, $n_1=1, n_2=2$.
   - $1/\lambda = R(1 - 1/4) = 3R/4 \Rightarrow \lambda = 4/3R$.

## Mathematics
5. **Answer: B (2)**
   - $n(A \cup B) = n(A) + n(B) - n(A \cap B)$.
   - $38 = 17 + 23 - n(A \cap B) \Rightarrow n(A \cap B) = 40 - 38 = 2$.

6. **Answer: A ($\{2, 4, 6\}$)**
   - $x + 2y = 8 \Rightarrow x = 8 - 2y$.
   - Since $y \in \mathbb{N}$ ($y \geq 1$):
     - If $y=1, x=6$.
     - If $y=2, x=4$.
     - If $y=3, x=2$.
     - If $y=4, x=0$ (not in $\mathbb{N}$).
   - Domain is $\{2, 4, 6\}$.

## Challenge Answer
7. **Proof Sketch:**
   - Bohr's quantization: $mvr = n\frac{h}{2\pi}$.
   - Rearrange: $2\pi r = n \frac{h}{mv}$.
   - Since $\lambda = h/mv$ (De-Broglie), $2\pi r = n\lambda$.
